# Аналіз частотного та байєсівського підходів до проведення та оцінки результатів А/В тестів

Цей проект досліджує частотний та байєсівський підходи до проведення та оцінювання результатів А/В тестів, використовуючи набір даних із BigQuery Public Datasets та синтетично створені сценарії для тестової групи (+20%, 0% та -20% відносно контрольної).

---

## Основні технології
- **Pandas**
- **Matplotlib**
- **scipy**
- **Теоретичні аспекти частотного та байєсівського підходів до проведення А/В тестування**

---

## Основні частини проекту

### 1. Вступ
- Короткий опис завдання проекту та основних аспектів двох підходів.

### 2. Основна частина дослідження
- Підготовка алгоритму для проведення А/В тестування за частотним підходом, перевірка роботи функцій великою кількістю симуляцій та порівняння отриманих результатів із теоретичними.
- Підготовка алгоритму для проведення А/В тестування за байєсівським підходом, перевірка роботи функцій великою кількістю симуляцій та порівняння отриманих результатів із частотним підходом. Підбір гіперпараметрів функції для оптимізації точності роботи алгоритму у сценарії із приблизно рівними конверсіями. 

### 3. Висновки
- Висновки щодо проведеного дослідження: порівняння результатів частотрного та байєсівського підходів для різних сценаріїв (+20%, 0% та -20% відносно контрольної), а також формулювання загальних думок щодо можливості використання того чи іншого підходу у бізнесі.   

---

Мова проекту - **Українська**.

Оскільки у проекті робота алгоритмів тестується значною кількістю симуляцій за підходом Монте-Карло, час виконання деяких функцій може бути значним.
